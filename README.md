# gosudoblock
Block users from running your app with root permissions

### How to install
```go
go get github.com/shamsher31/gosudoblock
```

### How to use
```go
package main

import (
  "fmt"
  "github.com/shamsher31/gosudoblock"
)

func main() {
   sudoblock()
}
```
